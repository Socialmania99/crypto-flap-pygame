**Crypto Flap** is a Pygame-based endless runner (flappy-style) game where you guide a Bitcoin through volatile Candlestick Pipes. It features a persistent local authentication system and a global/personal high score leaderboard saved to a local JSON file.

### Key Features:
* **Authentication Flow:** Seamless login/register screen that leads directly into gameplay.
* **Persistent Data:** Saves user records and high scores in `leaderboard.json`.
* **Adaptive UI:** Supports dynamic resizing and automatically switches layout between Portrait (Mobile style) and Landscape (Desktop style) orientations.
* **Progressive Difficulty:** Pipe speed and density increase as the player's score rises.
